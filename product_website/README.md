# Vortex Guru Product Website

Static marketing site concept for the Vortex Guru volumetric avatar platform. Built with vanilla HTML, CSS, and JavaScript—no build tooling required.

## Structure
- `index.html` — Single-page layout covering hero, feature pillars, industry solutions, storyboard, case studies, roadmap, FAQ, and contact form.
- `styles.css` — Dark neon-inspired visual theme, responsive layout, and component styling.
- `script.js` — Navigation toggle, scroll-based section highlighting, interactive storyboard, and auto-advancing case study slider.

## Preview
Open `index.html` in any modern browser. No additional setup is necessary.
